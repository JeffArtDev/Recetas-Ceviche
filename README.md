# Proyecto Ceviche

Proyecto de documentación breve sobre el **ceviche peruano**.

Este repositorio presenta de forma simple el concepto del ceviche, sus elementos principales y su importancia cultural, usando únicamente un archivo README.

---

## Objetivo
Documentar el ceviche como plato representativo del Perú en un formato claro y conciso.

---

## Contenido
- Definición general
- Ingredientes principales
- Contexto cultural
